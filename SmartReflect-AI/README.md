# SmartReflect AI 🚗✨

## AI-Based Real-Time Road Visibility Monitoring System

SmartReflect AI is a vehicle-mounted system that uses AI and computer vision to monitor road visibility in real time. It detects low-visibility road markings and helps authorities take timely maintenance actions.

---

## 🚀 Features

- Real-time road visibility detection  
- AI-based reflectivity analysis  
- GPS-based location tracking  
- Live dashboard with alerts  
- Simulation mode for demo  

---

## 🧠 Technology Stack

- Python (FastAPI)
- OpenCV
- HTML, CSS, JavaScript
- Chart.js
- SQLite

---

## ⚙️ How It Works

1. Capture road images using a camera  
2. Process images using OpenCV  
3. Estimate visibility levels  
4. Tag with GPS location  
5. Display on dashboard with alerts
